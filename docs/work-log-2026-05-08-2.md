## 주요 결정사항

### 영상 비용 정책 (확정)
- **길이**: 60초 cap (트리머 UI + 입력 검증)
- **크기**: 50MB cap (4K 거대 원본 방어)
- **만료**: 90일 lazy cleanup (트레이너 마운트 시 1회 / 24h gate)
- **압축**: 클라이언트 측 1Mbps bitrate + MediaRecorder 트림

### 인증 정책 (확정)
- **OAuth-only** — Google / Kakao 만. 전화 OTP 미사용
- 이유: SMS 한국 단가 ~$0.05/건 → 월 $35~50 부담. 또한 OAuth 쪽이 CR 더 높음

### 데이터 무결성 정책 (확정)
- 모든 storage 업로드는 **DB insert 와 페어** 로 묶음 — 한쪽 실패 시 반드시 롤백
- 회원 삭제 시 storage cascade 도 **DB cascade 직전** 에 호출 (cascade 후엔 path 정보 사라짐)
- 일지 발송은 **partial state 가 보이지 않게** 모든 단계 묶음 트랜잭션처럼 처리

### COEP 정책
- `credentialless` 채택 — Supabase storage 의 cross-origin 이미지 표시 우선
- COOP 는 `same-origin` 유지 (보안 격리)
