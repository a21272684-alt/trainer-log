## 4. 기존 트레이너/회원 자동 연동

OAuth 로그인 시 이메일 매칭으로 기존 레코드에 `auth_id` 자동 연결:

```js
// auth_id 없이 email만 있는 기존 레코드
await supabase.from('trainers').update({ auth_id: au.id }).eq('id', byEmail.id)
```
