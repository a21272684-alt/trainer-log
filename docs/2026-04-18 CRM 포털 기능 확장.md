### 2-3. 락커 / 운동복 대여 관리

**마이그레이션**: `trainer-log/supabase/migrations/020_locker_rentals.sql`  
**UI**: `MembersTab.jsx` → 회원 상세 "대여 현황" 탭

```sql
locker_rentals (
  rental_type   text  -- 'locker' | 'uniform'
  locker_number text  -- 락커 번호
  uniform_size  text  -- XS / S / M / L / XL / XXL
  start_date    date
  end_date      date
  returned_at   timestamptz
)
```

- 대여 등록 폼 (종류·번호/사이즈·메모)
- 반납 버튼 → `returned_at` + `end_date` 업데이트

### 2-4. 수업 예약 관리 탭

**파일**: `src/components/tabs/ScheduleTab.jsx`  
**마이그레이션**: `trainer-log/supabase/migrations/022_class_schedules.sql`

```sql
class_schedules (class_type, title, start_at, end_at, max_capacity, color, is_cancelled)
class_bookings  (schedule_id, member_id, status: confirmed|waitlist|cancelled|attended)
```

#### 기능
- **주간 뷰**: 월~일 7컬럼 캘린더 그리드, 날짜별 수업 카드
- **목록 뷰**: 전체 수업 테이블 (일시·종류·트레이너·예약/정원·상태)
- **수업 등록**: 종류·트레이너·정원·제목·시작/종료·색상 선택
- **수업 상세**: 예약 회원 목록, 회원 추가 예약, 개별 예약 취소, 수업 취소 처리
- **주 이동**: 이전/다음 주 이동, 오늘 버튼

### 2-5. 전자 계약서 탭

**파일**: `src/components/tabs/ContractsTab.jsx`  
**마이그레이션**: `trainer-log/supabase/migrations/021_member_contracts.sql`

```sql
member_contracts (
  contract_type  text  -- pt | group | membership
  product_name   text
  session_count  int
  amount         numeric
  start_date     date
  end_date       date
  special_terms  text
  signature_url  text  -- Supabase Storage 서명 이미지
  signed_at      timestamptz
  status         text  -- draft | signed | cancelled
)
```

#### 기능
- **계약서 초안 작성**: 회원·트레이너·종류·상품·기간·특약 입력
- **손서명 UI (Canvas)**: 마우스/터치 드로잉, 초기화, 서명 완료
- 서명 완료 → Supabase Storage(`contract-signatures`) 업로드 → URL 저장
- 계약서 목록: 초안/서명완료/취소 필터
- 계약서 상세: 서명 이미지 인라인 표시, 계약 취소

### 2-6. 알림 발송 탭

**파일**: `src/components/tabs/NotificationsTab.jsx`

#### 대상 자동 분류 (4개 유형)

| 유형 | 조건 |
|------|------|
| ⚠️ 만료 임박 | 잔여 세션 1~3회 |
| 🔴 세션 소진 | 잔여 세션 0회 |
| 📉 이탈 위험 | risk_level = risk or critical |
| 🏃 장기 미출석 | 마지막 출석 14일 이상 전 |

#### 기능
- 유형 탭 + 해당 회원 수 뱃지
- 체크박스로 개별/전체 회원 선택
- 기본 메시지 자동 삽입 (유형별 차별화)
- 발송 버튼 (SMS 연동 시 실제 발송, 현재는 이력 기록)
- 발송 기록 히스토리 (우측 패널, 최근 20건)
