## 1. 회원 포털 — 개인운동 탭 추가 (`5fc0b58`)

### 기능 개요
- 회원 포털에 **`🏃 개인운동`** 탭 추가 (전체 탭: 수업일지 / 건강기록 / 식단기록 / 개인운동 / 커뮤니티)
- 운동 세션을 회원이 직접 기록하고 조회

### 운동 기록 구성
| 항목 | 내용 |
|------|------|
| 날짜 | date 입력 |
| 운동 제목 | 텍스트 (예: "가슴·삼두 데이") |
| 운동 시간 | 분 단위 |
| 종목별 세트 | 종목명 + 세트 목록 (무게 / 횟수 / 휴식) |
| 메모 | 자유 입력 |
| 볼륨 자동 계산 | 무게 × 횟수 합계 |

### 근육 다이어그램 (MuscleDiagram 컴포넌트)
- 운동 세션 펼치면 앞면 / 뒷면 SVG 신체 이미지 표시
- 주동근 → 진한 색, 보조근 → 연한 색 (50% 투명도)
- 근육별 색상표:
  - 가슴 `#ef4444` / 등 `#3b82f6` / 어깨 `#8b5cf6`
  - 이두 `#f97316` / 삼두 `#06b6d4` / 하체 `#22c55e`
  - 코어 `#eab308` / 유산소 `#ec4899` / 전신 `#6b7280`

### 운동 자동완성 (`src/lib/exercises.js` 신규 생성)
- 60+ 종목 DB (`EXERCISE_DB`): 이름 / 주동근 / 보조근 / 장비
- 종목 입력 시 자동완성 드롭다운 → 선택 시 근육 자동 설정
- 루틴 저장 / 불러오기 기능

### DB 마이그레이션
- `012_workout_logs.sql` — `workout_sessions`, `workout_routines` 테이블 생성, RLS 전체 허용

## 2. 회원 포털 — 커뮤니티 탭 추가 (`5fc0b58`)

### 기능 개요
- 같은 트레이너 소속 회원끼리만 볼 수 있는 **인트라넷형 커뮤니티**
- 사진 + 이모지 반응 기능

### 구성
- 게시글 작성: 텍스트 + 사진 업로드 (Supabase Storage)
- 이모지 반응: `❤️ 🔥 💪 👏 😮 💯 🙌` (중복 토글)
- 반응 카운트 실시간 표시
- 게시글 삭제 (본인만)

### 사용 테이블
> ⚠️ `community_posts`는 CommunityPortal이 이미 사용 중 → 충돌 방지를 위해 별도 테이블 생성

| 테이블 | 용도 |
|--------|------|
| `member_posts` | 회원 커뮤니티 게시글 |
| `member_reactions` | 게시글별 이모지 반응 |

- `member_posts.trainer_id`로 같은 트레이너 소속 회원 게시글만 조회
- `community-photos` Storage 버킷 사용

## 4. 회원 개인운동 — 트레이너 열람 기능 (`3c68500`)

### 배경
- 회원이 기록한 운동 세션을 트레이너가 회원 상세에서 확인 가능하도록 추가

### 구현
- `workout_sessions.source` 컬럼 추가 (`'trainer'` | `'member'`)
  - 회원이 기록한 세션: `source: 'member'`
  - 트레이너가 기록한 세션: `source: 'trainer'`
- TrainerApp 회원 상세 → 개인운동 탭에서 세션 목록 조회
- **"회원 기록" 초록 배지** 표시 (회원이 직접 기록한 세션 구분)

```jsx
{s.source === 'member' && (
  <span style={{ background:'rgba(34,197,94,0.12)', color:'#16a34a' }}>회원 기록</span>
)}
```

### DB 마이그레이션
- `014_workout_source.sql` — `workout_sessions.source TEXT DEFAULT 'trainer'` 컬럼 추가

- `post_reactions` → `member_reactions`으로 교체
