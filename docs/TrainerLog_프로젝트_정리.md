# TrainerLog 프로젝트 정리

> 퍼스널 트레이너를 위한 수업일지 자동화 + 회원 관리 플랫폼

---

## 1. 프로젝트 개요

### 목적
- 트레이너가 매 수업마다 수업일지를 수작업으로 작성하는 불편함 해소
- AI를 활용해 음성 녹음 → 수업일지 자동 생성 → 회원에게 자동 발송
- 나아가 다른 트레이너에게 구독제로 서비스 판매 (SaaS화)

### 최종 구성 파일
| 파일 | 역할 | 접속 대상 |
|---|---|---|
| `trainer-app.html` | 수업일지 작성 · 회원 관리 | 트레이너 |
| `member-portal.html` | 수업일지 조회 · 체중/식단 기록 | 회원 |
| `admin-portal.html` | 전체 트레이너/회원/결제 관리 | 운영자(트레이너님) |

### 배포 주소
| 페이지 | URL |
|---|---|
| 트레이너 앱 | `trainer-log.vercel.app/trainer-app.html` |
| 회원 포털 | `trainer-log.vercel.app/member-portal.html` |
| 관리자 포털 | `trainer-log.vercel.app/admin-portal.html` |

## 2. 기술 스택

| 항목 | 선택 | 이유 |
|---|---|---|
| 프론트엔드 | HTML/CSS/JS 단일 파일 | 코딩 없이 유지보수 가능 |
| AI 요약 | Google Gemini 1.5 Flash API | 무료 플랜으로 개인 사용 가능 |
| 데이터베이스 | Supabase (PostgreSQL) | 무료 · 클라우드 · 실시간 |
| 배포 | Vercel | 무료 · GitHub 연동 자동 배포 |
| PDF 생성 | jsPDF | 수업일지 다운로드 |
| 차트 | Chart.js | 체중 변화 그래프 |
| 카카오톡 발송 | 클립보드 복사 + 앱 딥링크 | 별도 API 없이 즉시 사용 가능 |

## 3. 주요 기능

### 트레이너 앱
- **트레이너 등록/로그인** — 이름 + 전화번호 뒷4자리
- **회원 추가** — 이름, 전화번호, 레슨목적(필수), 총 세션 수, 메모
- **음성 녹음 업로드** — 아이폰 음성 메모 앱(m4a/mp3/wav)으로 50분 녹음 후 업로드
- **AI 수업일지 생성** — Gemini가 녹음 분석 → 중복 제거 → 자동 요약
- **운동 기록** — 운동 종목 / 세트&횟수 / RIR / 부족한 점·느낀 점
- **RIR 가이드** — 2RIR(복합운동) / 1RIR(대근육) / 0RIR(단관절) 참고표 내장
- **카카오톡 발송** — 수업일지 자동 복사 + 카카오톡 앱 열기
- **회원 건강기록 조회** — 체중 변화, 목표달성률, 식단기록 트레이너가 열람

### 회원 포털
- **로그인** — 이름 + 전화번호 뒷4자리 (트레이너가 등록한 정보로)
- **수업일지 탭** — 본인 일지 조회, PDF 다운로드, 복사
- **체중관리 탭**
  - 목표/시작/현재 체중 + 감량률 + 목표달성률 표시
  - 14일 공복 체중 변화 그래프 (목표 체중 기준선 포함)
  - 매일 공복 체중(아침) / 저녁 체중 / 수면레벨(1~10) 입력
  - 일별 일중증가 자동 계산
- **식단기록 탭** — 아침/점심/저녁/간식 분리 입력 · 히스토리 조회

### 관리자 포털 (운영자 전용)
- **대시보드** — 전체 트레이너/회원/수업일지/결제 현황 요약
- **트레이너 목록** — 가입일시, 회원수, 일지 발송건수, 구독상태, 상세보기
- **회원 현황** — 트레이너별 회원 목록, 레슨목적, 세션 진행률
- **수업일지 현황** — 오늘/이번주/이번달 발송 건수 탭 전환, 마지막 발송 시간
- **구독 관리** — 결제수단(카카오페이/카드/계좌이체/현금), 금액, 결제일, 만료일 관리
- 관리자 초기 비밀번호: `trainer2024!`

## 4. 데이터베이스 구조 (Supabase)

### 테이블 목록
```
trainers         — 트레이너 계정
members          — 회원 정보
logs             — 수업일지
health_records   — 체중/식단 기록
subscriptions    — 구독/결제 내역
```

### 주요 컬럼
**members 테이블**
- `name`, `phone` — 로그인용
- `lesson_purpose` — 레슨목적 (체형교정/근비대/다이어트/체력향상/재활/스포츠퍼포먼스/유지관리/기타)
- `total_sessions`, `done_sessions` — 세션 관리
- `target_weight`, `start_weight` — 체중 목표
- `age`, `height`, `special_note` — 프로필

**health_records 테이블**
- `morning_weight` — 공복 체중
- `evening_weight` — 저녁 체중
- `sleep_level` — 수면레벨 (1~10)
- `diet_note` — 식단 기록

## 7. 아이폰 사용 방법

### 수업일지 작성 흐름
1. 수업 중 아이폰 **음성 메모** 앱으로 50분 녹음
2. 수업 후 트레이너 앱 접속 → 회원 선택
3. 녹음 파일 업로드
4. **AI 수업일지 생성** 클릭
5. 내용 확인 후 **카카오톡으로 발송**
6. 세션 카운트 자동 +1, Supabase에 저장

### 음성 메모 파일 꺼내는 법
음성 메모 앱 → 녹음 선택 → 우상단 `···` → **파일로 내보내기** → 트레이너 앱에서 업로드
