3. "나머지 섹션도 히어로 톤" → wrapper 라임 + 섹션 투명
4. "모서리 라임 강조" → radial 코너 집중
5. "4코너 다" → 4 radial
6. "최상단/최하단만 적용?" → 모바일 fixed 미지원 설명 → fixed 오버레이 div 로 근본 해결
