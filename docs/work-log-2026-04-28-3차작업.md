## 2. AdminPortal.jsx — 기능 관리 탭 추가

- 사이드 네비에 `🔐 기능 관리` 탭 추가
- `FEATURE_DEFS` 상수 — 9개 기능 정의
- `DEFAULT_FEATURE_GATES` 상수 — 무료/유료 기본값
- `featureGates` state + `loadAll()` 연동 (`app_settings.feature_gates` 키)
- `saveFeatureGates()` — upsert 즉시 저장
- `toggleGate(plan, key)` — ON/OFF 토글
- `setMemberLimit(plan, val)` / `saveMemberLimit()` — 회원 수 제한

### 기능 관리 UI 구성
- **기능 토글 테이블**: 기능명 | 설명 | 무료 ON/OFF | 유료 ON/OFF
- **회원 수 제한 카드**: 무료/유료 숫자 입력 + 저장
- **현재 설정 요약 카드**: 플랜별 모든 기능 ON/OFF 한눈에 확인

## 3. TrainerApp.jsx — 기능 게이트 적용

### 추가된 로직

```js
const DEFAULT_FEATURE_GATES = { free: {...}, paid: {...} }
const [featureGates, setFeatureGates] = useState(DEFAULT_FEATURE_GATES)
const [isPaid, setIsPaid] = useState(false)
function canUse(key) {
  const plan = isPaid ? 'paid' : 'free'
  return featureGates[plan]?.[key] !== false
}
```

### `loadFeatureGates(trainerId)` — 로그인 시 자동 실행
- `subscriptions` 테이블에서 유효한 구독 확인 → `isPaid` 설정
- `app_settings.feature_gates` 로드 → `featureGates` 설정

### 적용된 기능 게이트 목록

| 기능 | 처리 방식 |
|------|----------|
| 발송기록·매출관리·시간표 탭 | 클릭 차단 + 🔒 뱃지 표시 |
| AI 수업일지 생성 버튼 | 잠금 UI 오버레이 (lock card) |
| 회원 추가 | member_limit 초과 시 toast 차단 |
| 주간 AI 리포트 | 잠금 UI (접기/펼치기 버튼 미표시) |
| 정산 분석 | 잠금 UI |
| AI 회원 인사이트 | 잠금 UI |
| 이탈위험 분석 필터 | 필터 버튼 자체 미표시 |
| Web Push 알림 설정 | 잠금 UI (설정 패널 미표시) |
