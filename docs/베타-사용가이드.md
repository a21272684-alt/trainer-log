# 오운(ownapp) 베타 사용 가이드

> 헬스장 트레이너와 회원이 함께 쓰는 운동 기록 앱 — 베타 버전입니다.
> 문의는 카카오톡 채널 **@ownapp** 또는 https://pf.kakao.com/_ownapp 으로 주세요.

**접속 주소:** https://ownapp.kr

---

## 처음 시작하시는 분께 — 30초 요약

오운은 두 가지 앱으로 나뉘어 있어요:

| 누가 | 어떤 앱 | 주소 |
|---|---|---|
| 트레이너 (PT 강사) | **트레이너 앱** | https://ownapp.kr/trainer |
| 회원 (PT 받는 분) | **회원 앱** | https://ownapp.kr/member |

**가입 방법:**
- **트레이너:** Google 로그인 → 이름 입력 → 가입 신청 → **관리자 승인 (평일 1~2일)** → 정식 사용
- **회원:** 트레이너가 먼저 본인을 회원으로 등록 → 회원이 Google 로그인 → 자동 진입

> 💡 **카카오 로그인은 현재 일시 비활성화 상태입니다.** Google 로그인만 사용해주세요. 정식 출시 단계에서 카카오 로그인이 추가됩니다.

# 🏋️ 트레이너 앱 사용법

## 1단계 — 가입하기

1. https://ownapp.kr/trainer 접속
2. **"Google로 로그인"** 클릭 → 본인 Google 계정 선택
3. **트레이너 등록** 화면에서:
   - 이름 입력 (앱에서 표시될 이름)
   - 이용약관 + 개인정보처리방침 동의 체크
   - **"트레이너 등록 완료"** 클릭
4. ⏳ **"가입 요청 승인 대기 중"** 화면이 뜨면 정상이에요.
5. 평일 기준 1~2일 안에 관리자가 승인합니다.
6. 승인 후 다시 https://ownapp.kr/trainer 접속해 Google 로그인하면 정상 진입.

> 📞 **승인 알림은 따로 보내지 않아요.** 며칠 후 다시 로그인하거나, 카카오 채널 **@ownapp** 으로 문의해 주세요.

## 2단계 — 메인 화면 6개 탭 한눈에

화면 아래쪽에 6개 탭이 있어요:

| 탭 | 하는 일 |
|---|---|
| 👥 **회원** | 회원 추가, 회원별 수업일지 작성, 출석/건강/홀딩(정지) 관리 |
| 📋 **히스토리** | 작성한 수업일지 전체 목록, 통계, 검색 |
| 📅 **일정** | 주간 스케줄 — 수업·개인 일정 블록 단위 관리 |
| 📊 **매출** | 세션 단가 자동 계산, 월별 정산, 주간 AI 리포트 |
| ⚙️ **설정** | 프로필, 알림, 플랜, 결제, 운영 정보 |
| 💬 **1:1 문의** | 카카오 채널로 바로 연결 (@ownapp) |

## 3단계 — 자주 쓰는 기능 사용법

### ✦ 회원 추가하기 (회원이 회원 앱 들어오려면 필수)
1. 👥 **회원** 탭 → 우측 상단 **+ 회원 추가** 버튼
2. 이름, **이메일 (Gmail 권장 — 회원 앱 OAuth 로그인용)**, 카카오 폰번호, 잔여 세션 수, 단가 등 입력
3. 저장
4. 회원에게 알림 — **"https://ownapp.kr/member 에 본인 Google 계정으로 로그인하시면 본인 정보가 자동으로 뜹니다"**

> 💡 회원의 이메일은 회원이 Google 로그인에 쓸 이메일과 정확히 일치해야 자동 매칭됩니다. 일치 안 하면 회원 앱 진입 불가.

### ✦ AI 수업일지 작성 (핵심 기능)
1. 👥 **회원** 탭에서 해당 회원 카드 클릭
2. **수업일지 작성** 화면에서 회차 입력
3. **🎤 녹음 시작** → 수업 진행하면서 녹음 (또는 수업 후 녹음 파일 업로드)
4. **AI 분석** 클릭 → 30초~1분 후 완성된 일지 (운동 종목, 세트, 코멘트) 자동 생성
5. 확인 + 수정 → **저장**
6. 회원에게 카카오톡으로 일지 링크 자동 발송 (옵션)

> 💡 녹음은 디바이스 (휴대폰/PC) 자체 음성인식으로 텍스트 변환됩니다. 외부 서버에 녹음 파일이 저장되지 않아요.

### ✦ 주간 리포트 (AI 자동 분석)
1. 📊 **매출** 탭 → 하단 **주간 리포트** 섹션
2. 매주 월요일 자동으로 직전 주 리포트가 대기 상태로 생성됨
3. **🤖 AI 리포트 지금 생성** 클릭 → AI 가 출석/매출/이탈위험/회원 추이 분석 → 완성된 리포트
4. **7일에 1회만** 생성 가능 (AI 비용 보호)
5. 과거 리포트는 탭으로 자유롭게 열람

### ✦ 매출 / 정산
- 회원별 세션 단가 기반으로 매월 자동 정산
- **대관 / 프리랜서 / 정직원** 3가지 고용 형태별로 다른 정산 방식 적용
- 월별 매출, 잔존 가치 (남은 세션 × 단가) 자동 계산

### ✦ 일정 관리
- 📅 **일정** 탭에서 주간 스케줄 블록으로 표시
- 수업 일정 + 개인 일정 함께 관리
- (정식 출시 후) 수업 30분 전 푸시 알림

### ✦ 회원 정지 (홀딩)
- 회원 카드 → **홀딩 탭** → 시작일/종료일/사유/사진 기록
- 회원 상태 자동 반영 → 정지 기간 동안 세션 차감 안 됨

### ✦ 출석 관리
- 회원 카드 → **출석 탭** → 캘린더에 체크
- 자동으로 세션 차감 + 잔여 세션 표시

### ✦ 식단 / 건강 정보 / 개인 운동 기록
- 회원 카드 안의 서브탭들로 진입
- 회원이 직접 회원 앱에서 기록하면 트레이너가 열람 가능

# 🧘 회원 앱 사용법

## 1단계 — 로그인하기

> ⚠️ **회원은 트레이너가 먼저 회원으로 등록해야 로그인 가능해요.**
> 등록 안 됐다면 트레이너에게 "오운 앱 회원으로 추가해주세요" 요청해주세요.

1. 트레이너에게 본인 정보 (이름, **Gmail 이메일**, 폰번호) 전달
2. 트레이너가 등록 완료 후 https://ownapp.kr/member 접속
3. **"Google로 로그인"** 클릭 → 트레이너에게 알려준 **그 이메일** 의 Google 계정 선택
4. 자동으로 본인 정보가 뜨면서 메인 화면 진입

> ❌ **로그인이 안 되고 "등록된 회원 정보가 없어요" 메시지가 뜨면:**
> - 트레이너가 입력한 이메일과 본인이 로그인한 이메일이 다른 경우예요.
> - 트레이너에게 "오운에 등록된 제 이메일이 OOO@gmail.com 맞나요?" 확인 요청.

## 2단계 — 메인 기능 4가지

### 📋 수업일지 열람
- 트레이너가 작성한 모든 수업일지 시간순으로 표시
- 운동 종목, 세트, 트레이너 코멘트 확인
- **PDF 저장** 버튼으로 일지 다운로드 가능

### ⚖️ 체중·건강 추적
- 매일 공복 체중 / 저녁 체중 / 수면 시간 기록
- 14일 추이 그래프 자동 표시
- 트레이너도 본인 동의 하에 열람 가능

### 🏃 개인 운동 일지
- 트레이너와 별개로 본인이 한 운동 기록
- **60+ 종목 자동완성** (벤치프레스, 스쿼트 등 입력하면 자동 추천)
- 세트별 무게·횟수·휴식 시간 기록
- **근육 다이어그램** 으로 어떤 부위 운동했는지 시각화
- 총 볼륨 (무게 × 횟수 × 세트) 자동 계산
- 트레이너가 만들어둔 **루틴 불러오기** 도 가능

### 🍱 식단 기록
- 아침 / 점심 / 저녁 / 간식 분류로 기록
- 음식 이름 입력 → **AI 음식 정보 자동 추천** (칼로리, 단백질/탄수화물/지방)
- 사진 업로드 → AI 가 음식 분석
- 자주 먹는 식단은 **템플릿 저장** → 다음에 1클릭 입력

### 🤝 회원 커뮤니티
- 같은 센터 회원들과 운동 일상 공유
- 사진 + 짧은 글로 게시물 작성
- 공지사항 / 자유게시판 두 종류

# ❓ 자주 묻는 질문 (FAQ)

### Q. 트레이너 가입 후 얼마나 기다려야 하나요?
A. 평일 기준 **1~2일** 안에 관리자가 승인합니다. 빠른 승인이 필요하면 카카오 채널 **@ownapp** 으로 문의해 주세요.

### Q. 카카오 로그인 버튼이 안 보여요.
A. 정상입니다. 베타 기간엔 **Google 로그인만** 사용해 주세요. 정식 출시 시점에 카카오 로그인이 추가됩니다.

### Q. Google 로그인 누르면 "액세스 차단됨" / "disallowed_useragent" 에러가 떠요.
A. **카카오톡 / 페이스북 / 인스타그램 등 SNS 앱 안의 인앱 브라우저에서 ownapp.kr 을 열었기 때문**입니다. Google 의 보안 정책으로 인앱 브라우저는 OAuth 로그인을 차단합니다.

해결 방법 (운영체제별):

**📱 안드로이드 폰**
1. 받은 링크를 길게 눌러 **"링크 복사"**
2. **Chrome 앱** 열고 주소창에 붙여넣기
3. Google 로그인 진행
> 또는 로그인 화면 상단의 노란 배너 **"🌐 Chrome으로 열기"** 버튼 클릭

**🍎 아이폰 (iPhone)**
1. 카카오톡 인앱 브라우저 우하단 **[Safari로 열기]** 또는 **[⋯]** 메뉴 → "Safari로 열기"
2. Safari 에서 Google 로그인 진행

**가장 확실한 방법:** Chrome 또는 Safari 주소창에 `ownapp.kr/member` 또는 `ownapp.kr/trainer` 직접 입력.

### Q. 회원이 회원 앱에 로그인 안 되는데요?
A. 두 가지 확인:
1. **트레이너가 회원으로 추가했는지** — 안 했으면 회원 앱 진입 불가
2. **이메일이 정확히 일치하는지** — 트레이너가 등록한 이메일과 회원이 Google 로그인한 이메일이 똑같아야 함

### Q. AI 수업일지가 작동하지 않아요.
A. 트레이너 앱에서 ⚙️ **설정** → **프로필** → AI API 키가 등록돼 있는지 확인. 정상이면 카카오 채널로 문의.

### Q. 주간 리포트 "재생성" 버튼이 회색이에요.
A. 정상입니다. AI 비용 보호를 위해 **7일에 1회만** 재생성 가능해요. 잠금 풀릴 때까지 남은 시간이 버튼에 표시됩니다.

### Q. 결제는 어떻게 하나요?
A. 베타 기간엔 **모든 기능이 무료** 입니다. 정식 출시 후 유료 플랜 (Pro, Premium) 안내드릴게요.

### Q. 가입을 취소하거나 데이터를 삭제하고 싶어요.
A. 카카오 채널 **@ownapp** 으로 연락 주세요. 24시간 안에 처리해드립니다.

### Q. 신규 기능 제안이나 버그 신고는 어디로?
A. 카카오 채널 **@ownapp** 또는 트레이너 앱의 💬 **1:1 문의** 탭. 베타 기간 피드백 1건당 무료 크레딧 추가 드려요.
