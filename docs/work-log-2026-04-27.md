**원인:** `row.value?.length` 조건이 빈 배열 `[]`에서 `0`(falsy)으로 평가되어 `setState([])` 호출이 스킵됨

```js
// Before (버그)
if (row.key === 'landing_reviews' && row.value?.length) setReviews(row.value)

// After (수정)
if (row.key === 'landing_reviews' && Array.isArray(row.value)) setReviews(row.value)
```

`landing_stats`, `landing_reviews`, `landing_kakao`, `landing_faqs` 4개 항목 모두 동일 패턴으로 수정

### 6. 랜딩페이지 스크롤 애니메이션

**위치:** `src/pages/Landing.jsx`  
**배경:** 헬스장부(healthjangbu.com) 스타일 참고, 스크롤 기반 동적 랜딩 구현

**추가된 애니메이션 3종**

#### FadeUp — 섹션 아래→위 페이드인
```jsx
function FadeUp({ children, delay = 0 }) {
  const [ref, inView] = useInView(0.1)
  return (
    <div ref={ref} style={{
      opacity: inView ? 1 : 0,
      transform: inView ? 'translateY(0px)' : 'translateY(52px)',
      transition: `opacity 0.85s cubic-bezier(.22,1,.36,1) ${delay}ms, ...`
    }}>{children}</div>
  )
}
```
- 적용: 전 섹션 헤더 (PROBLEM / SOLUTION / RESULTS / PRICING 등)

#### SlideCard — 카드 왼쪽→오른쪽 순차 슬라이드
```jsx
function SlideCard({ children, delay = 0 }) { ... }
// delay: i * 130ms 간격으로 순서대로 등장
```
- 적용: 문제/솔루션/후기/타겟/회원기능/카카오/요금제 카드 전체

#### CountUp — 숫자 카운팅 애니메이션
```jsx
function CountUp({ value }) {
  // "98%" → 0% ... 47% ... 98% (1.6s, ease-out cubic)
}
```
- 적용: 상단 신뢰 지표 숫자 (`3분`, `98%`, `0원`)
- `IntersectionObserver` 기반 — 뷰포트 진입 시 1회만 실행

**커밋:** `6082477`
